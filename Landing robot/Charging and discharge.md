
```python
# Conversion from battery percentage to kilometers
def convert_battery_to_km(battery_mAh, discharge_rate):
    return battery_mAh * discharge_rate

# Charging battery over time
def charge_battery(time, charging_rate):
    return time * charging_rate

# Conversion from kilometers to time
def convert_km_to_time(distance, speed):
    return distance / speed
```
